# 看護師勤務表管理システム v3.0

看護師の勤務表作成・管理を効率化するWebアプリケーション（Supabase統合版）

## 機能

- 👥 職員管理（追加・編集・削除）
- 🔑 アクセスコード発行　
- 📅 休み希望入力（職員用画面）
- 📋 希望確認・編集（管理者画面）
- 📊 前月Excel読み込み・制約適用
- 🤖 勤務表自動生成（週ごとの夜勤パターン対応）
- 📈 個人別・日別統計表示
- 📥 Excel出力
- 🖥️ 最大化表示

## 技術スタック

- React 18 + TypeScript + Vite
- Tailwind CSS
- Supabase（データベース）
- Vercel（デプロイ）

## ローカル開発

```bash
npm install
npm run dev
```
